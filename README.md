## Folder Structure

For smaller project like this.

```
/
├── index.html
├── assets/   |
    ├── css/
    │   ├── style.css
    │   └── ...
    ├── img/
    │   ├── image1.jpg
    │   └── image2.jpg
    │   └── image3.jpg
    │   └── ...
    ├── js/
    │   ├── script.js
    │   └── ...
    ├── fonts/
    │   ├── font.js
    │   └── ...
├── sass/
│   ├── main.scss
│   └── ...
└── ...
```

For larger Projects

```
/
├── index.html
├── src/
│   └── ...
├── dist or build/
│   └── ...
```
